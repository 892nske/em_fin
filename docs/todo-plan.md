# ToDoアプリ 実装ToDoリスト

## ゴール
- Must要件をすべて実装し、Should要件から少なくとも3項目（検索、設定保持、一括操作）を完了する。
- ローカルストレージによるデータ永続化とアクセシビリティ要件を満たす単一ページ体験を提供する。
- 自動テストカバレッジ90%以上、主要ユーザーストーリー8件の受け入れテスト合格。
